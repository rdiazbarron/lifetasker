# LifeTasker

LifeTasker es una aplicación web de productividad personal basada en bloques semanales flexibles.

La idea principal no es gestionar tareas diarias, sino objetivos semanales por bloques de actividad.

Ejemplos:
- Trabajo ESAM: 30 minutos
- Aprendizaje Tech: 30 minutos
- Ejercicio Cardio: 45 minutos

## Stack previsto

- Frontend: Next.js App Router
- Backend: NestJS
- Base de datos: PostgreSQL
- ORM: Prisma
- UI: TailwindCSS

## Concepto central

El usuario define cuántos bloques quiere completar durante la semana y puede completarlos en cualquier momento, sin asignarlos obligatoriamente a un día específico.

Ejemplo:

- Trabajo ESAM: 0/10
- Aprendizaje Tech: 0/2
- Ejercicio Cardio: 0/3
